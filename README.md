##从DOS和Unix开始谈现在操作系统的恩怨情仇

众所周知，如今我们使用的比较著名的三大操作系统有微软创建的windows系统，苹果公司开发Mac os系统的和linus开发的linux系统。那么这三种系统是如何被开发出来的？它们的背后又有什么关系以及故事呢？

说到这三种系统我们不得不提三个对整个世界都颇具影响力的传奇人物。那便是微软的创始人比尔盖茨，苹果的创建者乔布斯，还有linux的开发者linus。那么这三位天才之间又有什么恩怨情仇呢？首先，我们不得不从比尔盖茨和乔布斯说起。1955年2月24日，乔布斯在美国硅谷出生。他是个孤儿，从小被人领养。同年的10月28日，比尔盖茨在美国出生，父亲是个律师，母亲是一名老师。在17岁的时候，盖茨卖掉了他的第一个电脑编程作品，一个时间表格系统，买主是他的高中学校。72年，乔布斯上大一，退学之后，在74年又被佛教吸引，去了印度旅行，剃度为僧。76年，他回了国，与朋友在自家破烂车库中搞研究，并创造了Apple 2个人电脑。因此也赚到了人生的第一桶金。79年，经过两年的努力他的苹果成为个人电脑的代名词，他成功的成为了百万富翁。而此时，身无分文的比尔盖茨还在他的地下室里艰苦的搞BASIC解码器。80年，比尔盖茨从IBM 公司赚到了他人生中的第一个五万。同年，Apple 3推出，苹果上市乔布斯身价上涨超过一千万美元，成为了华尔街最年轻的富翁。96年，苹果公司宣布破产，2000年，乔布斯成为苹果永久的CEO。随后的10年他推出了改变世界的一系列苹果产品：iMac一体电脑，iPod一代和二代，iPhone3G及iPad平板电脑。2011年，乔布斯宣布辞职。但是这时候，我们都知道，苹果早已把诺基亚打压的半死不活，毫无疑问，苹果已经改变了手机的概念，改变了人类的生活。


比尔盖茨与乔布斯第一次见面是在77年西海岸电脑展。乔布斯在展厅的正门口高价租了一个展位，推出苹果公司的第二代产品Apple 2。他礼貌的向乔布斯伸出了手，说你好，我是比尔盖茨。当时乔布斯被一堆人簇拥着，享受着当明星的感觉，所以他根本没有看盖茨一眼。朋友问到说：那是比尔盖茨啊！乔布斯说到，是啊，不过，这是我人生中的第一次，是别人来找我，而不是我去找别人。但是他万万没想到，十几年之后，这位戴着厚镜片，不起眼的小个子将以胜过他几百倍的方式以其人之道还治其人之身。

第二次见面是微软开发了Excel，Word,Chart和File。比尔盖茨问乔布斯是否有兴趣，但是乔布斯抚摸着心爱的Macintosh,再一次置盖茨于不顾。随即，盖茨说到：“如果你不要的话，那我將卖给IBM”。乔布斯迟疑了一下，他无奈地发现在IBM和微软之间，他已经没有选择。80年，在IBM负责软件开发的工程师给微软盖茨的办公室打了一个电话，对微软说他们需要两样东西：程序语言和操作系统。程序语言，盖茨有Basic，操作系统他并没有拥有。盖茨想到艾伦有一个朋友，蒂姆帕特森，有一个操作系统，但无人购买，如今正在苦恼。于是艾伦便买了那个操作系统的所有权而这个操作系统便是DOS，windows的前身。85年，乔布斯30岁生日，恰恰是在这一年，他被公司罢免。就在他辞职之际，比尔盖 茨的图形界面出世了。盖茨拥有这个系统的所有权，并积极的贩卖给各个电脑厂商，终于，盖茨登上了世界首富的宝座。而这个系统的名字便是windows。
87年到97年比尔盖茨是这个是年的唯一明星。微软已雄霸天下。而苹果呢。96年苹果亏损10亿，且在几任外来者手里，苹果公司丧失了所有使它如此成功的创新能力。12月，乔布斯回归。这时，乔布斯明白，他一生的敌人，正是和苹果打着官司的比尔盖茨。97年，乔布斯引诱盖茨用1.5亿美元买了苹果公司的非投票股票。这笔钱，拯救了危在旦夕的苹果。苹果作为回报，在每一部Macintosh上内置了InternetExplorer和office。97年波士顿MacWorldExpo苹果产品展览会，乔布斯重出江湖。宣布与微软的合作。乔布斯仅仅用一年时间，就让苹果转亏为赢。98年，苹果退出笔记本iMac。2001年iPod和iTune横空出世，乔布斯把苹果公司带到音乐的世界。07年乔布斯退出触摸屏智能手机iPhone,将苹果带到手机业。2010年iPad 和iPhone4 。2011年iPhone4s发布会乔布斯没有参加，第二天便永远离开。一代伟人的离世，是对我们整个世界的损失，作为一名小小的后辈在此致敬！

linus,1969年出生在芬兰。Linux和Git这两个伟大的开源项目的创始人。他开发linux系统其实并不是什么伟大的构想。只是因为当时他想玩，而他又不喜欢DOS,unix又太贵，他 便自己研发了一个类unix系统，即linux系统。Linus从未见过微软公司的比尔盖茨，但却在2000年的时候遇上了乔布斯。那时候的他任职于美国半导体设计公司全美达。乔布斯邀请linus去位于库比蒂诺的苹果总部工作，为他提供丰厚的薪水和优越的职位，但是他在苹果的工作却是与Linux 无关的。因为这个原因，linus拒绝了乔布斯的邀请。况且，linus根本就不喜欢苹果操作系统的内核Mach。在那个时候，如果linus接受了邀请，答应去苹果工作，那现在的世界很可能会不一样。我们不会有Linux,Kindle和Android。有时候，linus由于他那些极具争议的言论而显得傲慢。但这与他对世界的贡献是不相符的。他创造了Linux和Git两个极好的开源项目，并使他们自由免费。有许多攻击linus的言论，说他没有拿出他的部分收入来改善这个世界。但事实上，linus作为Linux内核的总设计师将自己的作品变为免费。他本可以从中获得数以亿计的收入。他将他所拥有的一切无私平等的奉献给了整个世界。

作为对世界极有贡献三位传奇人物，我们不得不被他们的才智，天赋，努力所折服。希望通过了解三大操作系统背后的故事，能与你我共勉！
